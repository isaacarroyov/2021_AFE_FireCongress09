## Software requirements
In order to execute the script and the Jupyter Notebook successfully is fundamental to be registered to the Google Earth Engine platform.  Here are some guides that could help ease this procedure :eyeglasses: :
* [Sign Up to the platform](https://earthengine.google.com/signup/) :pencil2:
* [Earth Engine Code Editor](https://developers.google.com/earth-engine/guides/playground) (JavaScript) :earth_africa:
* [Python Installation](https://developers.google.com/earth-engine/guides/python_install) (Python) :snake:

The Python environment has the following libraries and versions:
* **Python**: 3.8.10
* **[JupyterLab](https://jupyter.org/install)** (`jupyterlab`): 3.0.16
* **GEE API** (`earthengine-api`): 0.1.272
* **[NumPy](https://numpy.org/install/)** (`numpy`): 1.20.2
* **[Pandas](https://pandas.pydata.org/docs/getting_started/install.html)** (`pandas`): 1.2.5
* **[Folium](https://python-visualization.github.io/folium/installing.html)** (`folium`): 0.12.0
* **[Matplotlib](https://matplotlib.org/stable/users/installing.html)** (`matplotlib`): 3.3.4
* **[Seaborn](https://seaborn.pydata.org/installing.html)** (`seaborn`): 0.11.1
* **[eemont](https://eemont.readthedocs.io/en/latest/#installation)** (`eemont`): 0.2.5
* **[geemap](https://geemap.org/installation/)** (`geemap`): 0.8.18
